# Automated Email-Triggered Dashboard

## Project Overview
This project demonstrates an automated dashboard that updates dynamically when data is received through email.

## How It Works
1. An email is received with updated data
2. Automation script extracts the attachment/content
3. Data is cleaned and processed
4. Dashboard refreshes automatically

## Tools & Technologies
- Python (Pandas)
- Power BI / Excel Dashboard
- Email Automation (SMTP / Power Automate)

## Dashboard Features
- Automatic refresh on new email data
- Real-time KPI updates
- Interactive visualizations

## Security Note
Sensitive information like email credentials is excluded for security reasons.
